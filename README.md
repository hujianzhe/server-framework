# util_test_client

1、底层库https://github.com/hujianzhe/util, 下载后置于BootServer目录下  
2、纯C实现，不依赖任何第三方框架  
3、跨平台服务器母机，可具体配置启动参数  
4、通过加载动态链接库实现不同的业务逻辑  
5、支持异步回调、协程、消息3种RPC调度机制  
6、网络通信支持TCP和可靠UDP，支持自定义消息编解码  
7、服务发现中心，集群节点建立  
8、支持按照一致性hash环，hash取余数，依次轮询三种方式选取目标节点
